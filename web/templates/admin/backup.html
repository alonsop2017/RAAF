Notes

  • pcr_credentials.yaml is excluded — it contains API secrets. Back it up manually if needed:
    cp config/pcr_credentials.yaml ~/pcr_credentials_backup.yaml
  • Local downloads are streamed in-memory — no temp files created on the server. The database is captured with a hot consistent snapshot so the zip is never mid-write.
  • LAN backups write to <path>/latest/ — rsync only copies changed files on subsequent runs. A backup_manifest.yaml in the backup directory records the timestamp of the last backup.
  • The directory is created if absent.
  • Local download filename format: raaf_backup_[quick|full]_YYYYMMDD_HHMMSS.zip
