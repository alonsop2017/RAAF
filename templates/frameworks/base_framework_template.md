## Customization Notes

When adapting this framework for a specific role:

1. **Adjust category weights** based on role priorities
2. **Modify criteria** within categories to match job requirements
3. **Update scoring guides** with role-specific benchmarks
4. **Add role-specific criteria** as needed
5. **Document all changes** in framework_notes.md

---

## Usage Instructions

1. Read the candidate's resume thoroughly
2. Score each criterion based on evidence found
3. Document specific evidence for each score
4. Calculate category subtotals and overall total
5. Apply recommendation threshold
6. Note key strengths, concerns, and interview focus areas
